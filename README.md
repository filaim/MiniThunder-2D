# MiniThunder 2D

2D-игра про танки, написанная на чистом JavaScript.

## Особенности
- Кампания из 10 уровней
- Режим «Шторм» с сужающейся зоной
- Бесконечный счёт и испытание на время
- Песочница для создания своих битв
- Кооперативное лобби (заготовка)
- Различное оружие (включая рельсотрон)
- Сохранение прогресса

## Запуск
Структура файлов:
"dist -> win-unpacked -> MiniThunder 2D.exe"
**ЛИБО**
Откройте `index.html` в браузере. Никаких серверов не требуется.

## Управление
- WASD (ЦФВЫ) / стрелки — движение
- Мышь — поворот башни и стрельба
- Q — смена режима рельсотрона (электро/огонь)
- Escape — пауза
- R — рестарт после проигрыша
