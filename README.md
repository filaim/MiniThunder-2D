# MiniThunder-2D
MiniThunder 2D — динамичные танковые бои, созданный с любовью к играм. 10 уровней кампании, режим «Шторм» с сужающейся зоной, бесконечный счёт, песочница, кооператив и арсенал оружия (включая рельсотрон). Сохранение прогресса, безумие противников и классные атаки боссов. Играй бесплатно прямо сейчас!
